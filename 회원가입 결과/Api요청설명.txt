1. 아이디 중복확인   URL : /auth/check-username?username="확인하려는아이디"  
Method : Get  
예시) http://localhost:8080/auth/check-username?username=testid
응답 : DB에 testid라는 유저아이디 존재하는 경우 문자열 "fault", HttpStatus409 반환
중복되지않는경우 문자열 "success", HttpStatus200 반환

2. 회원가입 URL: /auth/register  
Method : Post
예시)  http://localhost:8080/auth/register
postman 기준  Body / raw / JSON으로 
{
    "username": "test223",
    "password": "tset1123", 
    "name": "t123",
    "phoneNumber": "test2123" 
}
응답 : 요청 시 DB 테이블에 해당 유저 정보 추가 및 문자열 "success", HttpStatus 201 반환, 테이블 안에 username이 중복되는 경우 문자열 "fault", HttpStatus409 반환

3. 로그인 URL : /auth/login  
Method : Post
예시)  http://localhost:8080/auth/login
postman 기준  Body / raw / JSON으로 
{
   "username": "test223",
   "password": "tset1123"
}
응답 :  DB에 해당 유저 데이터 일치하는지 확인, 로그인 성공 시  문자열 "Login successful",    실패 시 "Invalid credentials" 반환

4. 비밀번호 변경  URL : /auth/change-password
Method : Post
예시) http://localhost:8080/auth/change-password
postman 기준  Body / raw / JSON으로 
{
    "username": "test223",
    "name": "t123",
    "phoneNumber": "test2123",
    "newPassword": "tset1123"
}
응답 : username, name, phoneNumber 3개가 일치할 경우 newPassword로 비밀번호 변경 
성공하면 문자열 "success"HttpStatus 200반환 , 실패 시 문자열 "user not found" HttpStatus404반환
